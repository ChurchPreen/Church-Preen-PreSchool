## Quick Setup (GitHub Pages)
1. Create a free GitHub account at github.com
2. Create a new PUBLIC repository (e.g. `website`)
3. Upload ALL these files keeping the folder structure
4. Go to Settings → Pages → Deploy from branch → main → Save
5. Site goes live at: `https://USERNAME.github.io/website/`

## Admin Panel
- URL: `yoursite/admin/`
- Default password: `ChurchPreen2025!` (change this!)
- You need a GitHub Personal Access Token (repo permission) to save changes
- Create one at: github.com/settings/tokens

## Adding Photos
Upload photos through the Admin Panel → Gallery section.
For section images (hero, team etc), upload them here in Claude and ask to embed them.

## Day-to-day Updates
