❓ Help & Handover Guide

Everything the next website manager needs to know.

🚀 Getting Started

  1. Go to github.com and log into the pre-school account
  2. Go to Settings → Developer settings → Personal access tokens
  3. Create a new token with repo permission — copy it
  4. Come to this admin page, enter the password and paste the token
  5. You are now able to save changes to the live website!

📰 Adding a News Post

Go to News in the sidebar → click Add News Post → fill in the title, date, and a short description → click Save Post. It will appear on the website within seconds.

📅 Updating Term Dates

Go to Term Dates → update the start, half term, and end dates for each term → click Save Term Dates.

📸 Adding Gallery Photos

Go to Gallery → click the upload area → select photos from your device. They will upload to the website automatically. To remove a photo, click the red ✕ on it.

🆘 Something went wrong?

If a save fails, check:
1. Your GitHub token is still valid (tokens expire — create a new one if needed)
2. The GitHub username and repository name in Settings are correct
3. You have internet access

Nothing you do in the admin panel can break the website permanently — every save is tracked by GitHub and can be undone.
